Engine, theme, content — never tangled.

  • Engine — scaling, navigation, presenter mode. Never touched.
  • Theme — colours, type, spacing, fonts. The whole look.
  • Content — your slides, one HTML file each.

Why it matters

Change any one layer without disturbing the others. Restyle an entire deck without editing a single slide.

Presentation ForgeThe idea

This separation is the core design bet. Slides only ever use the theme's vocabulary, which is exactly why themes are swappable.
